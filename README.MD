Датасет - https://www.microsoft.com/en-us/download/details.aspx?id=54765 

Набор файлов для предсказания кошек и собак на фото.
Сначала скачиваем датасет, структурно в 

    data
  
        test
  
            cats 
  
            dogs

Для train, test, val.

Можно легко сделать через программу split.py (предварительно очистив cleaner.py от битых изображений).
Затем настраиваем параметры в main2.py
После запускаем experiments.py, автоматически рассчитываются чекпоинты и сохраняется лучшая модель.
select_model.py показывает статистику помоделям
