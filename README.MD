# Cats vs Dogs Classifier

Классический проект по классификации изображений «Кошки vs Собаки».  
Репозиторий включает полный ML-pipeline:

- Подготовка датасета
- Очистка битых изображений
- Автоматическое разбиение на train/val/test
- Динамическая сверточная нейросеть (DynamicConvNet)
- Обучение с чекпоинтами
- Сохранение лучшей модели
- Визуализация результатов (графики, confusion matrix)
- Анализ и сравнение нескольких моделей

Проект полностью готов для запуска и расширения.

Датасет - https://www.microsoft.com/en-us/download/details.aspx?id=54765 

## Подготовка и запуск

1. **Очистка и разбиение датасета**  
   Сначала очистите битые изображения и разберите датасет на `train/val/test` с помощью скриптов:

   ```bash
   python tools/cleaner.py --input /path/to/images
   python tools/split.py --input /path/to/images --output data


Структура

    data
  
        test
  
            cats 
  
            dogs

Для train, test, val.

## Краткий workflow

1. **Очистка датасета**  
   Используйте `cleaner.py`, чтобы удалить битые изображения и подготовить папку с исходными данными.

2. **Автоматическое разбиение**  
   Запустите `split.py`, указав путь к необработанным изображениям и директорию для структурированного датасета (`train/val/test`).

3. **Настройка обучения**  
   Отредактируйте параметры в `main2.py`: пути, гиперпараметры, архитектуру модели.

4. **Запуск экспериментов**  
   Выполните `experiments.py` — создаст чекпоинты, сохранит лучшую модель и построит графики.

5. **Анализ моделей**  
   Откройте `select_model.py`, чтобы сравнить модели, посмотреть метрики и выбрать оптимальную.
